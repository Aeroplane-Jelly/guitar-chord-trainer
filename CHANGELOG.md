# Changelog

## [1.0.0] - 2026-04-10

### Initial release

- Random chord trainer for all 12 major keys
- Built-in metronome with adjustable BPM (20–300) and beats per chord
- Visual guitar fretboard diagrams with barre chord detection (50+ fingerings)
- Roman numeral display to reinforce music theory
- Difficulty modes: Beginner (open chords), Intermediate (open + barre), Advanced (all), Custom
- Custom chord selection — checkboxes to pick exactly which chords to practice
- Tap Tempo — set BPM by feel
- Session timer — auto-stop after 5, 10, 15, 20, or 30 minutes
- Volume control with +/− buttons
- Sound options — Default, Wood Block, Click, Bell
- Custom sound support via `sounds.json`
- Chord history — shows last 3 chords above the current chord
- 3-second countdown before session starts
- Progress tracking — sessions saved to `~/guitar_trainer_progress.json`
- Scrollable UI — content scrolls on smaller screens
- Launches fullscreen on startup
- Dark theme
