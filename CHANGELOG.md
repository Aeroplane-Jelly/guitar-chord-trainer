# Changelog

## [1.1.0] - 2026-04-10

### Added
- **Tap Tempo** — tap a button to set BPM by feel
- **Difficulty modes** — Beginner (open chords only), Intermediate (open + barre), Advanced (all), Custom
- **Custom chord selection** — checkboxes to pick exactly which chords from the key to practice
- **Session timer** — auto-stop after 5, 10, 15, 20, or 30 minutes
- **Countdown** — 3-second visual countdown before session starts
- **Chord history** — shows the last 3 chords played above the current chord
- **Sound options** — choose between Default, Wood Block, Click, and Bell click tracks
- **Progress tracking** — sessions saved automatically to `~/guitar_trainer_progress.json`

### Changed
- Window resized to accommodate new controls
- Session card added below settings for difficulty, timer, and sound options

## [1.0.0] - 2026-04-10

### Initial release
- Random chord trainer for all 12 major keys
- Built-in metronome with adjustable BPM and beats per chord
- Guitar fretboard chord diagrams with barre chord detection
- Roman numeral display
- Dark theme UI
